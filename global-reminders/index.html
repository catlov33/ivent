Расписание напоминаний о глобалах

Собери JSON для бота: время ивента, за сколько минут до него пинг, канал и роли. Экспортируй файл и загрузи в Discord командой /global_reminders_import.

Подробная инструкция (как пользоваться, роли, JSON, GitHub Pages)

Зачем этот сайт

Бот bot_2 по расписанию отправляет в выбранный текстовый канал сообщение с пингом ролей (например, глобал в 18:00 — напоминание в 17:00). Здесь ты настраиваешь правила визуально и скачиваешь global_reminders_import.json.

Файл roles.json

Список ролей с поиском подгружается из файла roles.json в этой же папке. Получить актуальный файл можно командой Discord /global_reminders_dump_roles (бот пришлёт вложение — сохрани как roles.json рядом с index.html в репозитории).

Сбор расписания

  1. Укажи часовой пояс по умолчанию (обычно Europe/Moscow).
  2. Отметь роли чекбоксами (или для каждой строки впиши ID ролей вручную).
  3. Для каждой строки: подпись, ID канала, время ивента, за сколько минут пинг, дни недели.
  4. Нажми «Скачать JSON» и отправь файл боту через /global_reminders_import.

Ограничение бота

Время напоминания должно попадать в тот же календарный день, что и ивент (например, нельзя поставить ивент на 00:30 и «за 90 минут» — напоминание ушло бы на прошлые сутки). Сайт проверяет это при экспорте.

Команды бота (кратко)

  • /global_reminder_add — одна запись без сайта
  • /global_reminder_list — список
  • /global_reminder_remove — удалить по номеру
  • /global_reminders_export — выгрузить текущее из бота
  • /global_reminders_dump_roles — roles.json для сайта
